# react-webpack-boilerplate

**`配置5小时开发2分钟`** 系列

- [x] webpack
- [x] webpack-dev-server
- [x] react
- [x] less
- [x] autoprefixer
- [x] px2rem


## USEAGE

```
git clone http://github.com/tomieric/react-webpack-boilerplate.git myproject && cd myproject
```

## start

```
npm start 
```

## build

```
npm run build
```
